# Admin Panel User Guide

## Accessing the Dashboard
Go to `/login` and enter your credentials. Once authenticated, you will be redirected to `/admin`.

## Content Management
- **Hero Section**: Update your name, role, and tagline.
- **Tech Stack**: Add or remove technologies. Use drag-and-drop to reorder.
- **Projects**: Manage your portfolio projects. Upload thumbnails and add tech tags.
- **Achievements**: Manage certificates and awards. Supports PDF uploads.

## Security
- Always log out when finished.
- Audit logs track all changes made in the admin panel.
